# Vjudge PDF Toolkit
### Generate combined pdf file from vjudge links.

------

## See the instructions before use.
### 1. Get a problem link from [Vjudge](https://vjudge.net/problem)
![alt text](https://raw.githubusercontent.com/minhaz1217/Vjudge-PDF-Toolkit/master/images/1.png "Step 1")
### 2. Open the program.
![alt text](https://raw.githubusercontent.com/minhaz1217/Vjudge-PDF-Toolkit/master/images/2.png "Step 2")
### 3. Put the vjudge link in the text box and click Add to add it in the list
![alt text](https://raw.githubusercontent.com/minhaz1217/Vjudge-PDF-Toolkit/master/images/3.png "Step 3")
### 4. Add several links to put them in the same PDF, to remove any link, double clik over the link
![alt text](https://raw.githubusercontent.com/minhaz1217/Vjudge-PDF-Toolkit/master/images/4.png "Step 4")
### 5. Click "Make PDF" and wait for the program to run.
![alt text](https://raw.githubusercontent.com/minhaz1217/Vjudge-PDF-Toolkit/master/images/5.png "Step 5")
### 6. A "GENERATION DONE" message will pop up to notify that the generation was successful.
![alt text](https://raw.githubusercontent.com/minhaz1217/Vjudge-PDF-Toolkit/master/images/6.png "Step 6")
### 7. The pdf will be Generated with name "Problem Set to Pdf" in the same folder as the exe file.
![alt text](https://raw.githubusercontent.com/minhaz1217/Vjudge-PDF-Toolkit/master/images/7.png "Step 7")
### 8. Open the PDF and verify whether the generation was done successfully.
![alt text](https://raw.githubusercontent.com/minhaz1217/Vjudge-PDF-Toolkit/master/images/8.png "Step 8")
### Refer to the How To Use to read these instructions again.
![alt text](https://raw.githubusercontent.com/minhaz1217/Vjudge-PDF-Toolkit/master/images/9.png "Help")
